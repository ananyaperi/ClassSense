# ClassSense
ClassSense: Autonomous classroom monitoring and assistance robot with noise hotspot detection, obstacle avoidance, and ESP32-hosted heatmap dashboard.

## Key Features
- Real-time noise monitoring  
- Autonomous navigation and obstacle avoidance  
- OLED display for reminder messages  
- Web-based heatmap visualization  
- ESP32 WiFi dashboard for robot control

## Hardware
ESP32 DevKit  
MAX9814 Microphone Module  
HC-SR04 Ultrasonic Sensor  
TB6612FNG Motor Driver  
DC Gear Motors  
SH1106 OLED Display
